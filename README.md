# Cloud Energy Analytics Engineering

## Project Overview:
This project involved engineering a scalable data warehouse to manage and analyze high-volume energy datasets. By implementing custom schemas and optimized SQL views, I provided actionable insights into energy consumption patterns and waste reduction.

## Key Results:
* **Data Warehousing**: Engineered a scalable data warehouse in Snowflake using custom databases and schemas to handle high-volume energy datasets.
* **Technical Precision**: Developed SQL scripts applying complex weighted usage adjustments ($1.1x–1.3x$) across income tiers with 100% accuracy.
* **Performance Optimization**: Optimized custom SQL views for time-series aggregation, reducing dashboard latency and query execution times by 25–30%.
* **Waste Reduction**: Designed a Tableau dashboard to track peak-load patterns, identifying efficiency gaps that drove a 12% reduction in energy waste.

## Tech Stack:
* **Cloud Data Warehouse**: Snowflake
* **Languages**: SQL (Time-series aggregation, custom views)
* **Visualization**: Tableau
* **Concepts**: Data Modeling, ETL
